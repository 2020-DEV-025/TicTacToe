## Tic Tac Toe
![Alt text](./public/android-chrome-512x512.png?raw=true "Tic Tac Toe")

## Run the game
To run the game locally, run:

```
npm install
npm start
```

Once it is done:
Open [http://localhost:3000](http://localhost:3000) to play on the browser.

## Run the tests
To test application, run:

```
npm test
```
